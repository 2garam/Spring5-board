# Spring MVC 미니 프로젝트(게시판)

[환경]
- 톰캣, 이클립스, java, Srping5, oracle 11g or mariaDB(클라우드 실구현)

[기능]
  1. 회원관리
    - 유효성 검사 처리
    - 아이디 중복확인 처리
    - 저장, 로그인 및 로그아웃
    - 정보 수정

  2. 게시글
    - 글 작성
    - 글 목록
    - 글 수정 ( 작성권한 포함 )
    - 글 삭제 및 페이징 처리

  3. 기타 에러메세지
